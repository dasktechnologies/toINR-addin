toINR Guide

Convert numbers to Indian Rupees in words instantly using the Indian Numbering System (Lakhs and Crores).

How to use:

=toINR.convertDigits(A1)

Examples:

  • 1,00,000 → One Lakh Rupees Only
  • 1,50,200.50 → One Lakh Fifty Thousand Two Hundred Rupees and Fifty Paisa Only
  • 10,00,00,000 → Ten Crore Rupees Only

Limits:

This add-in currently supports values up to 999,999,99,99,99,999 (999 Lakh Crores).
